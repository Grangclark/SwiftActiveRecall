**Q. Swiftで変数と定数を宣言する方法は？**

<思い出してから下を確認>

<details>
<summary>答えを見る</summary>

- `var` を使うと **変数**（値を変更できる）
- `let` を使うと **定数**（値を変更できない）

``` swift ```
var name = "Taro"
let age = 25
